**Carbon Accounting & ESG Compliance Engine**
*Origin: Simora Carbon → evolved into the core of the Sustain platform*
*Status: MVP Live (Sprint 1 complete)*

**What it does:**
Every organisation needs to measure, report and reduce its carbon footprint. CarbonSense is the engine that makes this possible — for any sector, any framework, any country.

**Core Features:**
```
DATA COLLECTION
├── OCR invoice scanning (electricity, gas, fuel, water, waste bills)
├── Excel / CSV import (ISO 14064 template)
├── Manual dynamic form (13 segment-specific field sets)
├── Supplier Scope 3 token portal (no account needed)
├── ERP connector (Phase 2: SAP, Logo, Oracle, Netsis)
└── Full audit trail (source + factor version + timestamp)

CALCULATION ENGINE
├── Scope 1 / 2 / 3 (GHG Protocol)
├── Market-based vs location-based Scope 2
├── PCAF (banks — financed emissions)
├── Product Carbon Footprint (ISO 14067)
├── CBAM declaration data
├── Emission factor versioning (TEİAŞ, DEFRA, DESNZ, IEA, IPCC AR6)
└── Proxy data engine (Scope 3 — CSRD Omnibus I compliant)

REPORTING FRAMEWORKS (10+)
├── TSRS 1 & 2 (Turkey — KGK)
├── CSRD / ESRS (EU — 50,000+ companies)
├── UK SRS / ISSB S1 & S2 (UK — FCA)

├── TCFD
├── ISO 14064-1:2018
├── ISO 14067 (Product Carbon Footprint)
├── CBAM (EU Carbon Border Adjustment Mechanism)
├── EU Taxonomy
└── SASB (all 77 industry volumes — 13 segments covered)

OUTPUT FORMATS
├── PDF Report (print-ready, regulator-grade)
├── Word / DOCX (editable)
├── PowerPoint / PPTX (board presentation — auto-generated)
├── Infographic (SVG/PNG — annual report, social media)
├── ESG Public Page (company.sustain.ai — live, embeddable)
├── Web Widget (iFrame embed for company website)
├── JSON API (Bloomberg ESG, MSCI integrations)
```
